The minorbsem package in R

2023/04/02

Tags: structural-equation-modeling Wishart MASEM misspecification minor-factors adventitious-error r-package

I’ve been working on the minorbsem package in R since the start of February. It’s my first proper attempt at an R package. The package allows you fit Bayesian SEMs while accounting for:

  1. the possibility that minor factors influence the correlation between items (Uanhoro, forthcoming), or
  2. adventitious error (Wu & Browne, 2015).

One can also fit meta-analytic SEMs under similar assumptions (Uanhoro, 2022).

The package is hosted on both GitHub and r-universe.dev. And the package website is https://jamesuanhoro.github.io/minorbsem/.

I plan to actively develop this package over time. Some of my future plans are in an Issue on the GitHub repo.

Also, there should be a tutorial paper in the near future on how to use the package.
